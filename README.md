

# 🍳 PantryPulse — Cook What You Have

PantryPulse is a smart recipe discovery web app that helps users cook meals using ingredients they already have. Simply add items from your pantry and get instant recipe suggestions with match percentages.

## 🚀 Features

* 🔍 Ingredient-based recipe search
* ⚡ Smart match percentage system
* 🧠 Autocomplete ingredient suggestions
* 🎯 Filters (Vegetarian, Quick, Easy, Spicy, Perfect Match)
* 📊 Nutrition info per recipe
* 📱 Fully responsive UI
* 💾 Local storage (persists pantry items)

## 🛠 Tech Stack

* HTML5
* CSS3 (Custom UI + Animations)
* Vanilla JavaScript
* LocalStorage API
